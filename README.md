# Call for Submission 诗歌征稿

## For Author作者须知

### Topics主题:

>All Poetry types are available for submission.
>主题不限

投稿语言不限

征稿日期：即日起至2026年2月1日。

所有被收录的诗歌在2026年3月均会分配DOI并永久储存且可以被检索。

>可选：所有被收录的投稿均可在2026年2月20日前提交poster展示或提交不超过15分钟的oral presentation.

### 投稿方式：

1. 邮箱投递：

可将word,pdf,LaTeX或其他形式的手稿发送至邮箱：20210057(at)ntit(dot)edu(dot)cn

>注：投稿邮箱需将(at)替换为'@'，将(dot)替换为'.'，此为反爬虫措施。

2. 网页投递：

将诗歌作品在页面上方Issue处新建Issue投递。

### Call for reviewer审稿人招募:

审稿人要求具备文学，艺术或其他相关领域博士学位，或拥有相关领域副高级职称。

申请需将个人简历，作品集或其他文件发送至投稿邮箱。
